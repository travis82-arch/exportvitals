# Oura PWA Dashboard (Starter)

This is a minimal, static PWA starter you can deploy on Cloudflare Pages.

## What's included
- `index.html` – UI shell
- `src/app.js` – basic app logic (local-only)
- `src/style.css` – simple styling
- `manifest.webmanifest` – PWA manifest
- `sw.js` – service worker for offline caching

## Local preview
Just open `index.html` in a browser, or run a simple web server.

## Deploy
Push to GitHub, then connect the repo in Cloudflare Pages.
No build step required.
